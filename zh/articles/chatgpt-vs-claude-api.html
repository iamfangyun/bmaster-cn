二、Prompt Caching：被忽视的省钱利器

很多人不知道，API 调用中有很大一部分 token 是重复的——比如系统提示词、文档上下文、代码库前缀。如果你用了 Prompt Caching，这些重复部分只需付正常价格的 10%-50%。

各模型的缓存折扣对比：

模型正常输入缓存读取折扣率

这意味着：如果你的应用有大量重复上下文（比如代码助手、RAG 系统），Claude 的缓存优势能大幅拉低实际成本。

四、订阅 vs API：什么时候该切换？

很多人默认订阅 ChatGPT Plus（$20/月），但如果你知道自己的实际 token 用量，API 可能更便宜。以 GPT-4o 为例：

五、选择建议

  • 日常对话/写作：DeepSeek V3 或 GPT-4o-mini，成本极低
  • 代码助手（高上下文）：Claude Sonnet 4 + Prompt Caching，综合成本最低
  • 复杂推理：o3（输入已降至 $2/1M，性价比大幅提升）
  • 批量数据处理：GPT-4o-mini 或 DeepSeek V3，便宜到可以忽略
  • 顶级质量需求：Claude Opus 4（贵但最强，适合关键任务）
